# Бьюти-коворкинги России 2026

Открытое исследование рынка бьюти-коворкингов России — от Marafet Research.

Лендинг с разбором рынка: размер, бизнес-модели, карта 30 игроков, регуляторика, поведение мастеров, зарубежные бенчмарки и тренды на 2026—2028.

## Что внутри

- Контур и размер рынка (1 500 локаций, 9 млрд ₽ оборота сегмента)
- 5 бизнес-моделей с разбором юнит-экономики
- TOP-30 игроков рынка с фильтрами
- 8 зарубежных бенчмарков (Sola Salon Studios, Booksy, Treatwell, Fresha и др.)
- Поведение мастеров и точки боли
- 3 регуляторных барьера 2026—2027
- Тренды и слабые места данных (стресс-тест выводов)

## Стек

Чистый HTML + CSS + JavaScript без зависимостей. Деплой через GitHub Pages.

## Локальный запуск

```bash
python3 -m http.server 4477
open http://localhost:4477
```

## Деплой на GitHub Pages

1. Создать публичный репозиторий на GitHub
2. Запушить содержимое директории
3. В Settings → Pages → Source выбрать ветку `main`, директория `/ (root)`
4. Через 1–2 минуты лендинг будет доступен по адресу `https://<username>.github.io/<repo>/`
